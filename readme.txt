Harmony is a Product Featured Audio Video Content plugin for WooCommerce. Powered by WordPress.

== Description ==

A Product Featured Audio Video Content plugin for WooCommerce. It also has support for the Dokan Multivendor plugin.

Harmony Video Content feature allows set video as a featured product. Its Audio Content feature allows adding sample audio on the Single Product page.

> You must need to install and activate the [**WooCommerce**](https://wordpress.org/plugins/woocommerce/) plugin to use this plugin.
If the [**Dokan Lite**](https://wordpress.org/plugins/dokan-lite/) plugin is activated, it will add its necessary fields for vendors. It does not require the Dokan Pro plugin.

Any special instructions do not require for the plugin. Just install and enjoy its feature. However, available features can be disabled from its setting's screen.
